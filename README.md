# 📚 Study Notes

논문, 기술 블로그, 유튜브 등을 정리하는 개인 학습 노트

## 📝 문서 작성법

### 1. VS Code 스니펫 사용 (추천)

마크다운 파일에서 `template-study-note` 스니펫 자동완성 입력

### 2. 템플릿 수동 복사

`template.md` 파일 내용을 복사해서 사용

### 3. 스크립트로 생성

```bash
./new-study.sh "제목" "카테고리"
# 예: ./new-study.sh "Redis 최적화" "database"
```

### 4. AI 활용

`.github/AI_PROMPT_GUIDE.md` 참고

```
이 템플릿 형식으로 [링크] 정리해줘
```
