## AI 요약

### 1. 아티클 개요

Gen Z 세대에게 소프트웨어 개발자 직업은 한때 "황금 티켓"이었다. 그러나 AI의 급속한 확산으로 주니어 개발자의 채용 시장, 학습 방식, 커리어 경로 전반이 근본적으로 변화하고 있다. 저자는 스스로를 "geriatric Zoomer(고령 줌머)"로 표현하며, 개인적 경험과 데이터를 결합하여 이 변화를 분석한다.

| 항목 | 내용 |
|------|------|
| 핵심 주제 | AI가 주니어 개발자 채용 시장과 커리어 경로에 미치는 영향 |
| 대상 세대 | Gen Z (특히 22~27세 주니어 개발자) |
| 핵심 메시지 | AI가 엔트리 레벨 직책을 대체하면서 주니어 개발자의 진입 장벽이 급격히 높아지고 있음 |
| 톤 | 개인 경험 + 데이터 기반 분석, 우려와 낙관의 균형 |

### 2. 주니어 개발자 채용 시장의 위기

AI 도구의 확산으로 기업들이 주니어 개발자 채용을 줄이고, AI로 대체하는 추세가 뚜렷하다.

│   ┌────────────────────────────────┐                            │

### 3. AI 도구 사용 현황과 학습 방식의 변화

AI가 학습 과정에서 "발견 단계(Discovery Phase)"를 제거하면서, 학생들이 즉각적인 답변에 의존하게 되고 핵심 기술이 위축되고 있다는 우려가 존재한다.

| 지표 | 수치 |
|------|------|
| 개발자 중 AI 도구 사용률 | 84% (2023년 대비 14% 증가) |
| 학생 중 교육 목적 AI 사용 경험 | 97% |
| 학습 목적 AI 사용 | 66% |
| AI 금지해도 사용하겠다는 학생 | 75% |
| AI 사용 학생의 시험 성적 향상 | +10% (Microsoft 연구) |
| AI로 작성된 논문 비율 | 11% (2억 개 논문 중, Turnitin) |

```
┌───────────────────────────────────────────────────────────┐
│           AI가 학습에 미치는 이중적 영향                     │
│                                                           │
│   긍정적 측면                  부정적 측면                  │
│   ┌──────────────────┐        ┌──────────────────────┐    │
│   │ 시험 성적 10% 향상│        │ "발견 단계" 제거      │    │
│   │ 학습 속도 가속    │ ◄───► │ 기초 기술 약화 우려   │    │
│   │ 접근성 향상       │        │ AI 의존도 심화        │    │
│   └──────────────────┘        └──────────────────────┘    │
│                                                           │
│   핵심 역설: 75%가 "ChatGPT 사용은 부정행위"라 인식하면서    │
│             동시에 75%가 "금지해도 사용하겠다"고 응답        │
└───────────────────────────────────────────────────────────┘
```

### 5. CS 학위의 가치 변화

미국의 CS 학위 수가 2011년 이후 2배 이상 증가했지만, 졸업생 실업률은 오히려 인문학 전공보다 높은 역설적 상황이 발생하고 있다.

| 전공 | 실업률 |
|------|--------|
| 컴퓨터 공학(CE) | 7.5% |
| 컴퓨터 과학(CS) | 6.1% |
| 인문학 | ~5.2% |
| 순수 미술 | < 7.5% |

## 내가 얻은 인사이트

### 인재 파이프라인 관점
1. **주니어-시니어 파이프라인 단절의 시한폭탄**
   - "주니어를 뽑지 않으면 시니어도 없다"는 단순하지만 강력한 경고다. 기업들이 단기 비용 절감을 위해 주니어 채용을 AI로 대체하면, 5~10년 후 시니어 인력 부족이라는 구조적 위기에 직면할 수 있다. 이는 개별 기업의 문제가 아니라 산업 전체의 인재 생태계 문제다.

2. **인턴십 소멸이 만드는 "경험의 사다리" 붕괴**
   - 엔트리 레벨 직책이 "2~5년 경력 요구"를 하면서 인턴십마저 줄어드는 상황은, 신규 진입자가 경험을 쌓을 수 있는 경로 자체가 사라지고 있음을 의미한다. 이 "닭과 달걀" 문제는 교육 기관, 기업, 정책 차원의 공동 대응이 필요하다.

### 학습 방식 변화 관점
3. **AI 시대의 "발견 학습(Discovery Learning)" 위기**
   - AI가 즉각적인 답변을 제공하면서 학습의 본질인 "시행착오를 통한 깊은 이해"가 약화되고 있다. 이는 단순히 교육 방법론의 문제가 아니라, 다음 세대 개발자의 문제 해결 능력 자체에 영향을 미칠 수 있는 구조적 변화다. 75%가 "부정행위라 인식하면서도 사용하겠다"는 역설은 이 문제의 심각성을 보여준다.

4. **CS 학위 가치의 역전 현상**
   - CS 졸업생 실업률이 인문학 졸업생보다 높다는 사실은 충격적이다. CS 학위 취득자가 2011년 대비 2배 이상 늘어난 공급 과잉과, AI로 인한 주니어 수요 감소가 겹치면서 발생한 구조적 미스매치다. 단순 코딩 능력만으로는 더 이상 차별화가 되지 않는 시대가 도래했음을 시사한다.

### 산업 구조 변화 관점
5. **"AI가 대체할 것 vs 새로 만들 것"의 불균형한 시차**
   - AI가 기존 역할을 대체하는 속도와 새로운 역할을 만들어내는 속도 사이에 시간적 갭이 존재한다. OpenAI의 $400K 콘텐츠 전략가 사례처럼 새로운 기회가 생겨나고 있지만, 이 전환기에 주니어 개발자들이 감당해야 할 불확실성의 비용은 세대적으로 불공평하게 분배되고 있다.
